# JavaScript

## Console Functions

```javascript
console.log() // log to the console

console.debug() // log debug messages

console.info() // log informative messages

console.warn() // log warning messages

console.error() // log error messages

console.assert() // log assertions (bool)

console.table() // log data in tables

// group console messages together
console.group()
console.groupEnd()

// measure code performance
console.time()
console.timeEnd()
```
